# 個人ブログ

ポートフォリオサイトのような大型ページではなく、ブログのみに特化したコンパクトで高速なページの構築挑戦する。

## [URL](https://blog.morimorig3.com/)

## 頑張ること

- 設計をしてコーディングをする（Figmaなどで画像を作る）
- Gatsbyのカスタマイズに挑戦する（画像を見やすく）

## 機能要件

- ブログ記事投稿機能
  - 画像
    - **サムネイル・クリックで拡大**
  - コードブロック
  - テーブル
